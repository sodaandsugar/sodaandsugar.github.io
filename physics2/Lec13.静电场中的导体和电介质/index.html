    Lec 2 静电场中的导体和电介质¶

    1 公式¶

    项目 公式
    电容 \(C=\frac{Q}{U_{\text{A}} - U_{\text{B}}}\)
    电容的串联 \(\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots + \frac{1}{C_n}\)
    电容的并联 \(C = C_1 + C_2 + \cdots + C_n\)
    电介质的极化强度 \(\boldsymbol{P}=\frac{\sum\boldsymbol{p}_{分子}}{\Delta V}=\chi_{e}\varepsilon_{0}\boldsymbol{E}=\epsilon_0(\epsilon_r-1)\boldsymbol{E}\)
    注：\(\chi_{\text{e}}\) 是电介质的极化率，与电场强度 \(\boldsymbol{E}\) 无关，仅与电介质的种类有关，是一个量纲为 \(1\) 的常量
    束缚电荷面密度 \(\sigma^{\prime}=P\cos\theta=P_{\text{n}}\)
    注：\(P_{\text{n}}\) 是电极化强度 \(\boldsymbol{P}\) 沿电介质表面外法线方向的分量，若 \(\theta<\frac{\pi}{2}\)，则 \(\sigma'\)为正，若 \(\theta>\frac{\pi}{2}\)，则 \(\sigma'\)为负
    电极化强度与束缚电荷分布的关系

    \(\oint_{S} \boldsymbol{P} \cdot \mathrm{d} \boldsymbol{S}=-\sum\limits_{(S_ \text {内})} q^{\prime}=-\int_{V} \rho^{\prime} \mathrm{d} V\)

    注：\(q^{\prime}\) 是闭合面 \(S\) 所包围的体积 \(V\) 内的束缚电荷，\(\rho^{\prime}\) 是体积元 \(\mathrm{d}V\) 处的束缚电荷体密度
    电介质中的场 \(\boldsymbol{E} = \boldsymbol{E}_0 + \boldsymbol{E}' =\frac{\sigma_0}{\varepsilon_0} - \frac{\sigma'}{\varepsilon_0}\)
    即：电介质中的场应是外电荷产生的场 \(\boldsymbol{E}_0\) 与束缚电荷产生的场 \(\boldsymbol{E}'\) 的矢量和
    注：一般来说，\(\boldsymbol{E}'\) 的大小和方向是逐点变化的，在均匀的电介质内部，\(\boldsymbol{E}'\) 和外电场 \(\boldsymbol{E}_0\) 的方向处处相反，这使得总电场 \(\boldsymbol{E}\) 比原来的 \(\boldsymbol{E}_0\) 弱
    电位移 \(\boldsymbol{D} = \varepsilon_0 \boldsymbol{E} + \boldsymbol{P} = (1 + \chi_e)\varepsilon_0 \boldsymbol{E} = \varepsilon_r \varepsilon_0 \boldsymbol{E} = \varepsilon\boldsymbol{E}\)
    注：电介质的相对介电常数 \(\varepsilon_r = 1 + \chi_e\)，介电常数 \(\varepsilon=\varepsilon_r \varepsilon_0 = (1 + \chi_e)\varepsilon_0\)
    注：上式仅在各向同性的电介质中且 \(\boldsymbol{E}\) 不很大的情况下才成立；在各向异性的电介质中，\(\boldsymbol{D}\) 与 \(\boldsymbol{E}\) 的方向一般是不同的
    电介质中的高斯定理 \(\oint_S \boldsymbol{D} \cdot d\boldsymbol{S} = \sum\limits_{(S_内)} q_0\)
    注：上式只包含自由电荷，不包含束缚电荷
    点电荷系统的能量 \(W = \frac{1}{2}\sum q_{i}U_{i} = \frac{1}{2}\iiint_{V} Udq = \frac{1}{2}\iiint_{V} U\rho dV = \frac{1}{2}\iint_{S} U\sigma dS\)
    注：\(U_i\) 为点电荷 \(q_{i}\) 所在处由 \(q_{i}\) 以外其它所有电荷所产生的电势
    带电导体的能量 \(W_{\text{e}} = \frac{1}{2}\frac{q^{2}}{C} = \frac{1}{2}CU^{2} = \frac{1}{2}qU\)
    注：\(U\) 是导体带电荷 \(q\) 时的电势
    带电电容器的能量 \(W = \frac{Q^{2}}{2C} = \frac{1}{2}C(U_A-U_B)^{2} = \frac{1}{2}Q(U_A-U_B)\)
    电场的能量密度 \(w_{\text{e}} = \frac{1}{2}\varepsilon_{0}\varepsilon_{\text{r}}E^{2} = \frac{1}{2}\varepsilon E^2=\frac{1}{2}DE\)
    电场的能量 \(W = \int_{V} w_{\text{e}} \mathrm{d}V = \int_{V}\frac{1}{2}\varepsilon_{0}\varepsilon_{\text{r}}E^{2}\text{d}V\)

    2 静电场中金属导体问题¶

    2. 1 静电平衡¶

    • 导体内部电场强度处处为零，即 \(\boldsymbol{E}=\boldsymbol{0}\)
    • 导体内部电势 \(\boldsymbol{V}\) 是常量，导体是个等势体，导体表面是个等势面
    • 导体表面上每一点处的电场强度处处与它的表面垂直，即 \(\boldsymbol{E}=\boldsymbol{E}_{\mathrm{n}}\)
    • 空心导体（导体空腔）上净电荷的分布完全与实心导体一样，即净电荷只能分布在外表面上，静电平衡时导体内部没有净电荷
    • 导体表面附近的电场强度为 \(E=\frac{\sigma}{\epsilon_0}\)
    • 孤立导体曲率越大的地方，\(\sigma\) 越大

    2. 2 静电屏蔽¶

    1. 空腔内无带电体：达到静电平衡时，空腔内的电场强度等于零，空腔内的物体或仪器不会受到外电场的影响
    2. 空腔内有带电体（导体空腔本身不带电）：
    情况 结果 示意图
    外表面不接地 外表面上电荷产生的电场会对外界产生影响
    外表面接地 外表面上无电荷，空腔内外的电场互不干扰
    常见操作 含义
    接地 改变导体所带电荷量，使得其电势为 0
    连接 两个导体等电势，电荷在两个导体间重新分布

    求感应电荷面密度，仅外表面带电，设未知数，内部电势为零求解
