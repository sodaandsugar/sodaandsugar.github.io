    11 玻尔的原子模型¶

    公式
    氢原子光谱的里德伯公式 \(\frac{1}{\lambda} = T(k)-T(n) = R_{\text{H}} \left( \frac{1}{k^2} - \frac{1}{n^2} \right)\)
    氢原子的玻尔模型 角动量 \(L_n = mvr = n \frac{h}{2\pi} = n \hbar\)
    跃迁频率 \(\nu_{nk} = \frac{E_n - E_k}{h} = \frac{me^4}{8\varepsilon_0^2h^3}\left(\frac{1}{k^2} - \frac{1}{n^2}\right)=\frac{\|E_1\|}{h}\left(\frac{1}{k^2} - \frac{1}{n^2}\right)\)
    轨道半径 \(r_n = n^2 \frac{\varepsilon_0 h^2}{\pi m e^2} = n^2 r_1\)，轨道速度 \(v_n = \frac{1}{n} \frac{e^2}{2\varepsilon_0 h}\)
    能级 \(E_n = -\frac{1}{n^2} \left( \frac{me^4}{8\varepsilon_0^2h^2} \right) = -\frac{13.6\ \text{eV}}{n^2}\)
    能级简并度 \(\sum_{l=0}^{n-1}(2l+1)=n^2\)
    取值范围 相关公式
    主量子数 \(n\) 决定原子能量（主要） \(E_n=-\frac{1}{n^2}\left(\frac{me^4}{8\varepsilon_0^2 h^2}\right)\)，\(n=1,2,3…\)
    角量子数（副量子数）\(l\) 决定轨道角动量大小 \(L=\sqrt{l(l+1)}\frac{h}{2\pi}\)，\(l=0,1,2,…,n-1\)
    磁量子数 \(m_l\) 决定轨道角动量空间取向（Z 轴分量） \(L_z=m_l\frac{h}{2\pi}\)，\(m_l=0,\pm1,\pm2,…,\pm l\)
    自旋磁量子数 \(m_s\) 决定自旋角动量空间取向 ——

    温馨提示

    玻尔理论中角动量最小值为 \(\frac{h}{2\pi}\)，量子力学中最小值为 \(0\)，实验验证量子力学正确

    公式
    自旋角动量 \(S=\sqrt{s(s+1)}\frac{h}{2\pi}\)，\(s=\frac{1}{2}\)
    自旋角动量 Z 轴分量 \(S_z=m_s\frac{h}{2\pi}\)，\(m_s=\pm\frac{1}{2}\)
    自旋磁矩 \(\vec{\mu}_s=-\frac{e}{m}\vec{S}\)
    • 玻尔磁子：轨道磁矩量子化最小值 \(\mu_B=\frac{eh}{4\pi m}=9.27×10^{-24}\ \text{A} · \text{m}^2\)

    定态薛定谔方程

    电子势能 \(U=-\frac{1}{4\pi\varepsilon_0}\frac{e^2}{r}\)，电子的定态薛定谔方程为 \(\frac{\partial^2 \psi}{\partial x^2}+\frac{\partial^2 \psi}{\partial y^2}+\frac{\partial^2 \psi}{\partial z^2}+\frac{2m}{\hbar^2}\left[E+\frac{e^2}{4\pi\varepsilon_0 r}\right]\psi=0\) 根据势场的球对称性，由 \(x=r\sin\theta\cos\varphi\)，\(y=r\sin\theta\sin\varphi\)，\(z=r\cos\theta\) 进行球坐标变换，得到：

    \[ \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial\psi}{\partial r}\right) + \frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial\psi}{\partial\theta}\right) + \frac{1}{r^2\sin^2\theta}\frac{\partial^2\psi}{\partial\phi^2} + \frac{2m}{\hbar^2}\left(E + \frac{e^2}{4\pi\varepsilon_0r}\right)\psi = 0 \]

    由于势能 \(U(r)\) 仅是 \(r\) 的函数，可用分离变量法求解，将波函数写作 \(\psi(r,\theta,\varphi)=R(r)\Theta(\theta)\Phi(\varphi)\)

    将此式代入上式，通过分离变量可得三个常微分方程：

    \[ \left\{ \begin{array}{l} \frac{d^2 \Phi}{d\varphi^2}+m_l^2\Phi=0 \\[6pt] \frac{1}{\sin\theta}\frac{d}{d\theta}\left(\sin\theta\frac{d\Theta}{d\theta}\right)+\left[l(l+1)-\frac{m_l^2}{\sin^2\theta}\right]\Theta=0 \\[6pt] \frac{1}{r^2}\frac{d}{d r}\left(r^2\frac{d R}{d r}\right)+\left[\frac{2m}{\hbar^2}\left(\frac{e^2}{4\pi\varepsilon_0 r}+E\right)-\frac{l(l+1)}{r^2}\right]R=0 \end{array} \right. \]
