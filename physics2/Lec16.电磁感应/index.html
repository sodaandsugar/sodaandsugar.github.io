    Lec 16 电磁感应¶

    楞次定律

    闭合回路中感应电流的方向，总是使感应电流所产生的通过回路面积的磁通量，去反抗引起感应电流的磁通量的变化

    1 基础公式¶

    公式
    法拉第定律 \(\varepsilon = -\frac{\text d (\sum_{i=1}^{N} \varPhi_i)}{\text{d}t} = -\frac{\text{d}\varPsi}{\text{d}t}\) 或 \(\varepsilon = -\frac{N \varPhi}{\text{d}t}\)
    感应电流 \(I_i = \frac{\varepsilon_i}{R} = -\frac{N}{R} \frac{\text{d}\varPhi}{\text{d}t}\)
    感生电荷 \(q = \int_{t_1}^{t_2} I_i \text{d}t = -\frac{N}{R} \int_{\varPhi_1}^{\varPhi_2} \text{d}\varPhi = \frac{N}{R} (\varPhi_1 - \varPhi_2)\)
    导体棒运动产生的动生电动势 \(\varepsilon = \int_{b}^{a} (\mathbf{v} \times \mathbf{B}) \cdot \text{d}\mathbf{l} = Blvsin \theta\)
    圆盘旋转产生的动生电动势 \(\varepsilon = \frac{1}{2}B l^2 \omega\)
    交流发电机 \(\varepsilon=NBS\omega\sin\left(\omega t\right)\)
    涡旋电场 \(\oint_{L} \mathbf{E}_{\text{旋}} \cdot \mathrm{d}\mathbf{l} = -\dfrac{\mathrm{d}\varPhi_{\mathrm{m}}}{\mathrm{d}t} = -\iint_{S} \frac{\partial \mathbf{B}}{\partial t} \cdot d\mathbf{S}\)
    感生电动势 \(\varepsilon = \int_{a}^{b} \vec{E}_{i} \cdot d\vec{l}\)
    自感系数 \(L = \frac{d \Phi}{d I}\)
    自感电动势 \(\varepsilon_L = -L \frac{\mathrm{d}I}{\mathrm{d}t}\)
    LR电路接通 \(I=\frac{\varepsilon}{R}\left(1-\mathrm{e}^{-\frac{R}{L}t}\right)\)
    注：弛豫时间 \(\tau=L/R\)
    LR电路切断 \(I=I_{0}\mathrm{e}^{-\frac{R}{L}t}\)
    互感电动势 \(\varepsilon_{21} = -M \frac{\mathrm{d}I_1}{\mathrm{d}t}\)，\(\varepsilon_{12} = -M \frac{\mathrm{d}I_2}{\mathrm{d}t}\)
    互感 \(M = \frac{N_2 \Phi_{21}}{I_1}\) 或 \(M = \frac{N_1 \Phi_{12}}{I_2}\)
    自感为\(L\)、通有电流\(I\)的线圈内的磁场能量 \(U_m = \frac{1}{2}LI^2\)
    磁场的能量密度 \(u_m = \frac{1}{2} \frac{B^2}{\mu_0 \mu_r} = \frac{1}{2}\mu_0 \mu_r H^2 = \frac{1}{2}HB\)
    磁场的能量 \(U_m = \int_V u_m \mathrm{d}V = \int_V \frac{1}{2}\mu_0 \mu_r H^2 \mathrm{d}V\)

    注释：对于这类问题，既可直接用法拉第电磁感应定律 求解总的感应电动势，也可分别计算动生电动势和感生 电动势：即假设磁场不变化，金属棒切割磁场线，求  动；再假设金属棒不动，由磁场变化，求 感。最后求 出总的感应电动势。
